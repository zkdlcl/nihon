## 🔴 Part 1: 조사 빈칸 채우기 (を / に / で / が / は)

다음 문장의 빈칸에 올바른 조사를 넣으세요.

1. テレビ ( ) 見ます。
2. 学校 ( ) ご飯を食べます。
3. 日本 ( ) 行きます。
4. 家 ( ) 近くに学校があります。
5. 犬 ( ) います。
6. 仕事 ( ) します。
7. 車の中 ( ) かばんがあります。
8. 音楽 ( ) 好きです。
9. お金 ( ) 欲しくないです。
10. コーヒー ( ) 飲みませんでした。

<details>
<summary>✅ 정답 보기</summary>

1. テレビ **を** 見ます。
2. 学校 **で** ご飯を食べます。(动作장소 → で)
3. 日本 **に** 行きます。(방향 → に/へ)
4. 家 **の** 近くに学校があります。(の는 보기에 없지만 조사 문제)
5. 犬 **が** います。(존재 → が)
6. 仕事 **を** します。(목적어 → を)
7. 車の中 **に** かばんがあります。(존재 장소 → に)
8. 音楽 **が** 好きです。(好き/嫌い/欲しい 앞 → が)
9. お金 **は** 欲しくないです。(주제 대조 → は)
10. コーヒー **を** 飲みませんでした。(목적어 → を)

</details>

## 🔴 Part 2: い형용사 시제 변환 (가장 자주 틀린 유형!)

다음 い형용사를 4가지 형태로 변환하세요.

| 기본형 | 현재 | 부정 | 과거 | 과거부정 |
|---|---|---|---|---|
| 安(やす)い (싸다) | | | | |
| 暑(あつ)い (덥다) | | | | |
| 忙(いそが)しい (바쁘다) | | | | |
| 面白(おもしろ)い (재미있다) | | | | |
| いい (좋다) ⚠️ | | | | |

<details>
<summary>✅ 정답 보기</summary>

| 기본형 | 현재 | 부정 | 과거 | 과거부정 |
|---|---|---|---|---|
| 安(やす)い | 安いです | 安**くない**です | 安**かった**です | 安**くなかった**です |
| 暑(あつ)い | 暑いです | 暑くないです | 暑かったです | 暑くなかったです |
| 忙しい | 忙しいです | 忙しくないです | 忙しかったです | 忙しくなかったです |
| 面白い | 面白いです | 面白くないです | 面白かったです | 面白くなかったです |
| いい ⚠️ | いいです | **よくない**です | **よかった**です | **よくなかった**です |

</details>

## 🔴 Part 3: な형용사 시제 변환

다음 な형용사를 4가지 형태로 변환하세요.

| 기본형 | 현재 | 부정 | 과거 | 과거부정 |
|---|---|---|---|---|
| 静(しず)か (조용하다) | | | | |
| 親切(しんせつ) (친절하다) | | | | |
| 暇(ひま) (한가하다) | | | | |
| 有名(ゆうめい) (유명하다) | | | | |
| 嫌(きら)い (싫어하다) ⚠️ | | | | |

<details>
<summary>✅ 정답 보기</summary>

| 기본형 | 현재 | 부정 | 과거 | 과거부정 |
|---|---|---|---|---|
| 静か | 静か**です** | 静か**じゃありません** | 静か**でした** | 静か**じゃありませんでした** |
| 親切 | 親切です | 親切じゃありません | 親切でした | 親切じゃありませんでした |
| 暇 | 暇です | 暇じゃありません | 暇でした | 暇じゃありませんでした |
| 有名 | 有名です | 有名じゃありません | 有名でした | 有名じゃありませんでした |
| 嫌い ⚠️ | 嫌いです | 嫌い**じゃありません** | 嫌いでした | 嫌いじゃありませんでした |

> ⚠️ **嫌い**는 い로 끝나지만 な형용사! `嫌いくないです` ❌

</details>

## 🟡 Part 4: 한국어 → 일본어 작문

다음 한국어 문장을 일본어로 작성하세요. (틀렸던 패턴 중심)

1. 오늘은 조금 바빴습니다. (いそがしい 과거형 주의!)
2. 그 선생님은 친절하지 않았습니다. (な형용사 과거부정)
3. 고양이가 책상 아래에 있습니다. (존재 + 위치)
4. 저는 벌레를 싫어합니다. (好き/嫌い 표현)
5. 은행과 편의점 사이에 우체국이 있습니다. (위치 종합)
6. 어제는 좋은 날씨였습니다. (いい의 불규칙 과거형)
7. 이것은 유명한 레스토랑입니다. (な형용사 + 명사 수식)

<details>
<summary>✅ 정답 보기</summary>

1. 今日(きょう)は 少し(すこし) 忙(いそが)しかったです。
2. その 先生(せんせい)は 親切(しんせつ)じゃありませんでした。
3. 猫(ねこ)が 机(つくえ)の 下(した)に います。
4. 私(わたし)は 虫(むし)が 嫌(きら)いです。
5. 銀行(ぎんこう)と コンビニの 間(あいだ)に 郵便局(ゆうびんきょく)が あります。
6. 昨日(きのう)は **よ**かったです。/ 昨日は いい天気(てんき)でした。
7. これは 有名(ゆうめい)**な** レストランです。

</details>

## 🟡 Part 5: 틀린 것 찾기 퀴즈

다음 문장에서 틀린 부분을 찾아 올바르게 고치세요.

1. `トマトがきれいです。` (토마토가 싫습니다 라는 의미로 작성)
2. `昨日は いそがしいでした。`
3. `静かくないです。`
4. `しずかまちです。`
5. `車の中でかばんがあります。`
6. `車が欲しです。`

<details>
<summary>✅ 정답 보기</summary>

1. `トマトが **きらい** です。` ← きれい(예쁘다) ≠ きらい(싫다)
2. `昨日は いそがし**かった**です。` ← い형용사 과거형은 かった
3. `静か**じゃない**です。` ← な형용사 부정형은 じゃない
4. `静か**な**町です。` ← な형용사 + **な** + 명사
5. `車の中**に**かばんがあります。` ← 존재 장소는 に
6. `車が欲し**い**です。` ← い형용사의 い 빠짐

</details>

## 🟢 Part 6: 자유 작문 연습

아래 주제로 일본어로 3~5문장을 자유롭게 작성해 보세요.  
(배운 표현을 최대한 많이 사용해 보세요!)

**주제**: 나의 방/집 소개
