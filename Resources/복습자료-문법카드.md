# 📋 문법 총정리 카드 (Day 1~12 학습 내용)

> Day 1~12까지 배운 모든 문법 포인트를 하나의 카드로 정리했습니다.  
> 이 페이지 하나만 보면 배운 내용 전체를 빠르게 훑어볼 수 있습니다.

---

## 📌 CARD 1: 핵심 조사 9개 총정리

| 조사 | 의미 | 사용 상황 | 예문 |
|---|---|---|---|
| **は** | ~은/는 (주제) | 이미 알고 있는 주제 | 私**は**学生です |
| **が** | ~이/가 (주격) | 새로운 정보, 강조, 존재, 好き/嫌い | 犬**が**います |
| **を** | ~을/를 (목적어) | 동사의 목적어 | 映画**を**見ます |
| **に** | ~에/에게 (목적지/위치) | 방향, 존재 장소, 시간 | 学校**に**います |
| **で** | ~에서/으로 (장소/수단) | 동작의 장소, 수단, 재료 | 学校**で**勉強します |
| **へ** | ~에/으로 (방향) | 이동 방향 | 日本**へ**行きます |
| **と** | ~와/과 (동반/열거) | 함께, 그리고 | 友達**と**行きます |
| **から** | ~부터/에서 (시작) | 시간·장소의 시작점 | 9時**から** |
| **まで** | ~까지 (끝) | 시간·장소의 끝점 | 5時**まで** |

## 📌 CARD 2: 명사문 시제 (です형)

```
[명사] + です/でした/じゃありません/じゃありませんでした
```

| 시제 | 표현 | 예문 |
|---|---|---|
| 현재 | **です** | 学生**です** (학생입니다) |
| 과거 | **でした** | 学生**でした** (학생이었습니다) |
| 현재 부정 | **じゃありません** | 学生**じゃありません** (학생이 아닙니다) |
| 과거 부정 | **じゃありませんでした** | 学生**じゃありませんでした** (학생이 아니었습니다) |

> 💡 회화체: `ではありません` → `じゃありません` (훨씬 자연스러움!)

## 📌 CARD 3: 동사 ます형 활용 (4단 변환)

```
[동사 ます형 기본형] + ます/ません/ました/ませんでした
```

| 시제 | 표현 | 예문 |
|---|---|---|
| 현재/미래 | **ます** | 食(た)べ**ます** (먹습니다) |
| 현재 부정 | **ません** | 食(た)べ**ません** (먹지 않습니다) |
| 과거 | **ました** | 食(た)べ**ました** (먹었습니다) |
| 과거 부정 | **ませんでした** | 食(た)べ**ませんでした** (먹지 않았습니다) |

### 동사 그룹 (ます형 변환 규칙)
| 그룹 | 변환 규칙 | 예시 |
|---|---|---|
| 1그룹 (う동사) | 어미를 い단으로 바꾸고 + ます | 書(か)く → 書(か)**き**ます |
| 2그룹 (る동사) | る를 빼고 + ます | 食(た)べる → 食(た)べ**ます** |
| 3그룹 (불규칙) | 통째로 암기 | する → **します** / 来(く)る → **来(き)ます** |

## 📌 CARD 4: い형용사 활용 (4단 변환)

```
[い형용사 기본형에서 い를 뺀 형태] + 활용 어미
```

| 시제 | 변환 | 예문 |
|---|---|---|
| 현재 | い + **です** | 高(たか)**い**です (비쌉니다) |
| 현재 부정 | い 탈락 + **くないです** | 高(たか)**くない**です (비싸지 않습니다) |
| 과거 | い 탈락 + **かったです** | 高(たか)**かった**です (비쌌습니다) |
| 과거 부정 | い 탈락 + **くなかったです** | 高(たか)**くなかった**です (비싸지 않았습니다) |

### ⚠️ 불규칙: いい (좋다)
| 시제 | 표현 |
|---|---|
| 현재 | いい / **よい**です |
| 현재 부정 | **よく**ないです |
| 과거 | **よかった**です |
| 과거 부정 | **よくなかった**です |

## 📌 CARD 5: な형용사 활용 (명사와 동일!)

```
[な형용사] + です/じゃありません/でした/じゃありませんでした
```

| 시제 | 표현 | 예문 |
|---|---|---|
| 현재 | **です** | 静(しず)か**です** (조용합니다) |
| 현재 부정 | **じゃありません** / じゃないです | 静か**じゃありません** |
| 과거 | **でした** | 静か**でした** (조용했습니다) |
| 과거 부정 | **じゃありませんでした** | 静か**じゃありませんでした** |

### ⚠️ 명사 수식 시: な형용사 + **な** + 명사
| ❌ 틀린 표현 | ✅ 올바른 표현 |
|---|---|
| 静か町 | 静か**な**町 |
| 有名レストラン | 有名**な**レストラン |

---

## 📌 CARD 6: い vs な형용사 한눈에 비교

| 구분 | い형용사 | な형용사 |
|---|---|---|
| 과거형 | ~**かったです** | ~**でした** |
| 부정형 | ~**くないです** | ~**じゃありません** |
| 명사 수식 | い 그대로 + 명사 | **な** + 명사 |
| 예 (과거) | 面白**かった**です | 静か**でした** |
| 예 (명사 수식) | 面白**い**映画 | 静か**な**町 |

### ⚠️ 예외 (い로 끝나지만 な형용사!)
| 단어 | 의미 | 부정형 |
|---|---|---|
| 嫌(きら)い | 싫어하다 | 嫌い**じゃありません** |
| きれい | 예쁘다/깨끗하다 | きれい**じゃありません** |

## 📌 CARD 7: 존재 표현 あります vs います

| 동사 | 사용 대상 | 예문 |
|---|---|---|
| **あります** | 무생물 (사물, 식물) | 本(ほん)が**あります** / 花(はな)が**あります** |
| **います** | 생물 (사람, 동물) | 犬(いぬ)が**います** / 先生(せんせい)が**います** |

### 위치 표현 세트
| 표현 | 의미 | 예문 |
|---|---|---|
| ~の 上(うえ) | ~의 위 | 机の上に本があります |
| ~の 下(した) | ~의 아래 | 椅子の下に猫がいます |
| ~の 前(まえ) | ~의 앞 | 駅の前にコンビニがあります |
| ~の 後ろ(うしろ) | ~의 뒤 | 学校の後ろに公園があります |
| ~の 右(みぎ) | ~의 오른쪽 | コンビニの右に銀行があります |
| ~の 左(ひだり) | ~의 왼쪽 | 駅の左に郵便局があります |
| ~の 間(あいだ) | ~의 사이 | AとBの間にCがあります |
| ~の 近(ちか)く | ~의 근처 | 家の近くに学校があります |

## 📌 CARD 8: 기호와 희망 표현

| 표현 | 의미 | 조사 | 예문 |
|---|---|---|---|
| ~が 好(す)きです | ~을/를 좋아합니다 | **が** | 音楽が**好き**です |
| ~が 嫌(きら)いです | ~을/를 싫어합니다 | **が** | 虫が**嫌い**です |
| ~が 欲(ほ)しいです | ~을/를 원합니다 | **が** | お金が**欲しい**です |
| ~が 大好(だいす)きです | ~을/를 정말 좋아합니다 | **が** | 猫が**大好き**です |

### 부정형
| 표현 | 의미 |
|---|---|
| 好きじゃありません | 좋아하지 않습니다 |
| 嫌いじゃありません | 싫어하지 않습니다 |
| 欲しく**ない**です | 원하지 않습니다 (い형용사!) |
