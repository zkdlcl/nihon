## 🟢 마스터한 영역 (Strengths)
*AI 튜터 노트: 이 부분은 사용자가 완벽하게 구사하므로 가볍게만 복습에 섞어주세요.*

- **존재의 조사**: `に`(존재 장소)와 `で`(동작 장소)의 명확한 구분
- **수단/방법 조사**: `で` (예: 車で行きます)
- **호불호/소망 조사**: `~が好き/嫌い/欲しい` 앞의 `が` 사용 완벽
- **명사 수식**: **な**형용사 + な + 명사 (`静かな町`)
- **구어체 부정**: `〜じゃないです`의 자연스러운 활용

## 🔴 아킬레스건 / 반복 취약점 (Weaknesses)
*AI 튜터/퀴즈 마스터 노트: 이 부분은 사용자가 무의식적으로 반복 실수하는 영역입니다. 다음 세션과 퀴즈에서 집중 타격(Drill) 하세요.*

### 1. い형용사 과거형의 환각 (이중 시제)
- **증상**: 과거형을 만들 때 `かった` 뒤에 자꾸 `でした`를 붙이려고 함.
- **❌ 오답 패턴**: `忙しいでした`, `忙しかったでした`
- **✅ 정답**: `忙しかったです`
- **처방**: い형용사가 보이면 무조건 `でした`를 금지시키는 규칙을 각인시킬 것.

### 2. な형용사의 과거 부정형
- **증상**: 부정형까지는 쓰지만, '과거의 부정'으로 넘어갈 때 꼬임.
- **❌ 오답 패턴**: `親切じゃないでした`
- **✅ 정답**: `親切じゃありませんでした` 또는 `親切じゃなかったです`
- **처방**: `じゃないでした`는 일본어에 없는 문법임을 반복 주지.

### 3. 고질적인 한자/어휘 혼동 (기억의 간섭)
- `きれい`(예쁘다) vs `きらい`(싫어하다) : 특히 `きらい`의 `い`를 빼먹는 실수 발현.
- `郵便局`(우체국): 郵 대신 자꾸 **湯**(뜨거운 물)를 씀. (湯便局 ❌)
- `天気`(날씨) vs `電気`(전기): 발음/한자 혼동.
