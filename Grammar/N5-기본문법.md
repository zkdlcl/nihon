# N5 기본 문법 정리

## 1. 기본 조사 (Particles)

### は (wa) - 주제 표시
- **용법**: 문장의 주제를 나타냄
- **예문**: 
  - 私**は**学生です。(저는 학생입니다)
  - これ**は**本です。(이것은 책입니다)

### が (ga) - 주격 조사
- **용법**: 문장의 주어를 강조
- **예문**:
  - 誰**が**来ましたか。(누가 왔습니까?)
  - 猫**が**います。(고양이가 있습니다)

### を (wo/o) - 목적격 조사
- **용법**: 동작의 대상
- **예문**:
  - ご飯**を**食べます。(밥을 먹습니다)
  - 本**を**読みます。(책을 읽습니다)

### に (ni) - 시간/장소/방향
- **용법**: 시간, 존재 장소, 이동 방향
- **예문**:
  - 7時**に**起きます。(7시에 일어납니다)
  - 部屋**に**います。(방에 있습니다)
  - 学校**に**行きます。(학교에 갑니다)

### で (de) - 장소/수단/방법
- **용법**: 동작이 일어나는 장소, 수단
- **예문**:
  - 図書館**で**勉強します。(도서관에서 공부합니다)
  - バス**で**行きます。(버스로 갑니다)

### へ (e) - 방향
- **용법**: 이동 방향 (に와 유사)
- **예문**:
  - 東京**へ**行きます。(도쿄로 갑니다)

### と (to) - 함께/인용
- **용법**: ~와 함께, 인용
- **예문**:
  - 友達**と**遊びます。(친구와 놉니다)
  - りんご**と**みかん (사과와 귤)

### から (kara) - 시작점/이유
- **용법**: 시간/장소의 시작, 이유
- **예문**:
  - 9時**から**始まります。(9시부터 시작합니다)
  - 東京**から**来ました。(도쿄에서 왔습니다)

### まで (made) - 종료점
- **용법**: 시간/장소의 끝
- **예문**:
  - 5時**まで**働きます。(5시까지 일합니다)

## 2. です/ます형

### です (desu) - 명사문
- **현재**: ~です (입니다)
- **과거**: ~でした (이었습니다)
- **부정**: ~ではありません / ~じゃありません (이 아닙니다)
- **과거부정**: ~ではありませんでした (이 아니었습니다)

**예문**:
- 学生です。(학생입니다)
- 学生でした。(학생이었습니다)
- 学生ではありません。(학생이 아닙니다)

### ます (masu) - 동사 정중형
- **현재/미래**: ~ます (합니다)
- **과거**: ~ました (했습니다)
- **부정**: ~ません (하지 않습니다)
- **과거부정**: ~ませんでした (하지 않았습니다)

**예문**:
- 食べます。(먹습니다)
- 食べました。(먹었습니다)
- 食べません。(먹지 않습니다)
- 食べませんでした。(먹지 않았습니다)

## 3. 형용사

### い형용사
- **현재**: ~いです
- **과거**: ~かったです
- **부정**: ~くないです / ~くありません
- **과거부정**: ~くなかったです

**예문** (大きい - 크다):
- 大きいです。(큽니다)
- 大きかったです。(컸습니다)
- 大きくないです。(크지 않습니다)

### な형용사
- **현재**: ~です
- **과거**: ~でした
- **부정**: ~ではありません
- **과거부정**: ~ではありませんでした

**예문** (きれい - 예쁘다):
- きれいです。(예쁩니다)
- きれいでした。(예뻤습니다)
- きれいではありません。(예쁘지 않습니다)

## 4. 기본 의문사

- **何 (なに/なん)**: 무엇
- **誰 (だれ)**: 누구
- **どこ**: 어디
- **いつ**: 언제
- **どう**: 어떻게
- **どれ**: 어느 것
- **いくら**: 얼마
- **なぜ/どうして**: 왜

## 5. 기본 문형

### ~があります/います (존재)
- **あります**: 무생물이 있다
- **います**: 생물이 있다

**예문**:
- 机の上に本**があります**。(책상 위에 책이 있습니다)
- 部屋に猫**がいます**。(방에 고양이가 있습니다)

### ~が好きです/嫌いです (좋아함/싫어함)
**예문**:
- 音楽**が好きです**。(음악을 좋아합니다)
- 野菜**が嫌いです**。(채소를 싫어합니다)

### ~が欲しいです (원함)
**예문**:
- 新しい車**が欲しいです**。(새 차를 원합니다)
