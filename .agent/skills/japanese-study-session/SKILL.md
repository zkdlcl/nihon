# Japanese Study Session Skill

이 스킬은 매일 일본어 학습 세션을 시작할 때 사용합니다.

## 실행 순서

### 1. 현재 진도 확인
- `Week1-2-계획.md` 파일을 열어 오늘이 몇 일차인지 확인
- 오늘의 학습 목표 체크리스트 확인

### 2. Daily Log 생성
- `Daily-Logs/` 폴더에 오늘 날짜로 새 파일 생성
- 형식: `YYYY-MM-DD-Day{N}.md`
- 템플릿 적용

### 3. 학습 시작 안내
사용자에게 다음 정보 제공:
- 오늘의 학습 목표 (문법/어휘/연습)
- 예상 소요 시간
- 학습 순서 제안

### 4. 대화형 학습 진행
- 문법 설명 및 예문 제공
- 어휘 학습 및 암기 팁
- 연습 문제 출제
- 실시간 피드백

### 5. 학습 완료 후
- 오늘 배운 내용 요약
- Daily Log에 기록
- 체크리스트 업데이트 안내
- Git commit 제안

## 사용 예시

사용자가 "일본어 공부 시작" 또는 "Day 1 시작" 등으로 호출하면:

1. 오늘의 학습 계획 표시
2. Daily Log 자동 생성
3. 첫 번째 학습 항목부터 시작
4. 대화형으로 진행

## 템플릿 위치
- Daily Log 템플릿: `Daily-Logs/로그-템플릿.md`
- 주간 계획: `Week1-2-계획.md`
- 문법 자료: `Grammar/`
- 어휘 자료: `Vocabulary/`
