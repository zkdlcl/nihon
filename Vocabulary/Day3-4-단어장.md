# Day 3-4 필수 어휘 리스트 (장소/물건/음식)

이번 주에 배운 조사(へ、から、まで)와 문법(です)을 활용하기 좋은 필수 단어들입니다.
소리 내어 읽으면서 외워보세요.

---

## 🏢 1. 장소 (Location)
> **へ (에) / から (부터) / まで (까지)** 랑 같이 쓰세요!

| 일본어 (한자) | 발음 (Hiragana) | 뜻 | 예문 |
| :--- | :--- | :--- | :--- |
| **学校 (がっこう)** | **갓코-** | 학교 | 学校へ行きます (학교에 갑니다) |
| **家 (うち / いえ)** | **우치 / 이에** | 집 | 家から (집에서부터) |
| **会社 (かいしゃ)** | **카이샤** | 회사 | 会社まで (회사까지) |
| **駅 (えき)** | **에키** | 역(Station) | 駅へ (역으로) |
| **病院 (びょういん)** | **뵤-인** | 병원 | 病院へ (병원에) |
| **トイレ** | **토이레** | 화장실 | トイレへ (화장실에) |
| **コンビニ** | **콘비니** | 편의점 | コンビニまで (편의점까지) |
| **食堂 (しょくどう)** | **쇼쿠도-** | 식당 | 食堂へ (식당으로) |
| **図書館 (としょかん)** | **토쇼칸** | 도서관 | 図書館から (도서관에서) |
| **部屋 (へや)** | **헤야** | 방 | 私の部屋 (내 방) |

## 📱 3. 물건 (Things)
> **これ (이것) / それ (그것) / あれ (저것) は ~ です** 로 연습하세요!

| 일본어 (한자) | 발음 (Hiragana) | 뜻 | 예문 |
| :--- | :--- | :--- | :--- |
| **本 (ほん)** | **홍** | 책 | 本です (책입니다) |
| **車 (くるま)** | **쿠루마** | 차(Car) | 車じゃありません (차가 아닙니다) |
| **かばん** | **카방** | 가방 | 私のかばんです (내 가방입니다) |
| **時計 (とけい)** | **토케이** | 시계 | いい時計ですね (좋은 시계네요) |
| **傘 (かさ)** | **카사** | 우산 | 傘です (우산입니다) |
| **携帯 (けいたい)** | **케-타이** | 휴대폰 | 携帯じゃありません (휴대폰이 아닙니다) |
| **靴 (くつ)** | **쿠츠** | 신발 | 赤い靴 (빨간 신발) |
| **眼鏡 (めがね)** | **메가네** | 안경 | 眼鏡です (안경입니다) |
| **財布 (さいふ)** | **사이후** | 지갑 | 私の財布 (내 지갑) |
| **鍵 (かぎ)** | **카기** | 열쇠 | 車の鍵 (차 열쇠) |

## 💡 암기 팁
1. **장음(길게 끄는 소리)** 주의: `코-히-`, `케-타이`, `토케이(토케-)`, `갓코-` 등
2. **연관성** 찾기: `우치`와 `이에`는 둘 다 '집'이지만, `우치`가 더 감성적인 '우리 집' 느낌입니다.
3. **가타카나**도 같이: `토이레`, `콘비니`, `팡`, `코-히-` 처럼 외래어는 가타카나로 씁니다.
