# 📖 한자 뜯어보기 마스터 파일

> 한자를 처음 배우는 분을 위한 파일이에요!  
> 한국어 한자음(음독)과 뜻을 함께 외우면 일본어 단어가 훨씬 빨리 외워져요.

---

## 📌 사용법

```
한자 = 뜻(한국 한자음)
예) 食 = 먹을 식

→ 食堂 = 먹을 식 + 집 당 = 식당 = しょくどう
→ 食べる = 먹다 = たべる
```

## 🎯 한자 퀴즈

뜻을 보고 한자를 맞혀보세요!

1. 배울 **학** + 교 **교** = ?
2. 전기 **전** + 말할 **화** = ?
3. 하늘 **공** + 항구 **항** = ?
4. 땅 **지** + 아래 **하** + 쇠 **철** = ?

<details>
<summary>정답 보기 👆</summary>

1. **学校** (がっこう) - 학교
2. **電話** (でんわ) - 전화
3. **空港** (くうこう) - 공항
4. **地下鉄** (ちかてつ) - 지하철

</details>

## 💡 암기 전략

1. **뜻 먼저** → 한자 뜯어서 의미 파악
2. **연결** → 한국어 단어에서 찾기 (학교, 식당, 전화...)
3. **읽기** → 일본어 음독/훈독 따로 외우기
4. **단어로** → 단어에서 자연스럽게 익히기
