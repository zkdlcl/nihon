# Day 2: 기본 조사 (を, に, で) 및 일상 동사

## 📅 날짜: 2026-02-15

### 학습 주제
- **문법**: 조사 を(을/를), に(에/에게), で(에서/으로)
- **어휘**: 일상 생활 동사 30개 (먹다, 마시다, 가다 등)
- **연습**: 조사 활용 문장 만들기

---

## 💬 대화 내용

### 나의 일본어
```
1. テレビ오見ます
2. cofee　오 のみます
3. 学校でご飯を食べます
4. 会社で仕事わします
5. 日本に行きます
6. 先生に会います
```

### AI 교정본
```
1. テレビを見ます。(텔레비전을 봅니다)
2. コーヒーを飲みます。(커피를 마십니다)
3. 学校でご飯を食べます。(학교에서 밥을 먹습니다) - **완벽합니다!**
4. 会社で仕事をします。(회사에서 일을 합니다)
5. 日本に行きます。(일본에 갑니다) - **완벽합니다!**
6. 先生に会います。(선생님을 만납니다) - **조사 선택이 아주 좋습니다!**
```

### 교정 포인트
1. **문법 오류**: 
   - 오류: `오` (한글 발음 표기)
   - 올바른 표현: `を` (히라가나)
   - 설명: 목적격 조사 '을/를'은 히라가나 `を`로 표기합니다. (키보드 입력: `wo`)

2. **어휘 선택**:
   - 사용한 단어: `cofee`
   - 올바른 표현: `コーヒー`
   - 설명: 외래어인 커피는 가타카나로 `コーヒー`라고 씁니다. 장음(-)에 주의하세요.

3. **기타**:
   - 동사 활용 `見ます`(봅니다), `飲みます`(마십니다)는 아주 정확하게 잘 쓰셨습니다!

4. **조사 표기 교정**:
   - 오류: `仕事わ` (wa)
   - 올바른 표현: `仕事を` (wo)
   - 설명: '일을'이라고 할 때의 조사는 `を`입니다. `わ`는 주로 감탄사나 여성어의 어미로 쓰이거나, 단어의 일부로 쓰입니다. 목적격 조사는 무조건 `を`를 씁니다.

## 📝 새로 배운 표현

| 표현 | 의미 | 예문 |
|------|------|------|
| テレビ | 텔레비전 | テレビを見ます (TV를 봅니다) |
| コーヒー | 커피 | コーヒーを飲みます (커피를 마십니다) |
| 見(み)ます | 봅니다 | 映画(えいが)を見ます (영화를 봅니다) |
| 飲(の)みます | 마십니다 | 水(みず)を飲みます (물을 마십니다) |
| 学校(がっこう) | 학교 | 学校で勉強します (학교에서 공부합니다) |
| 会社(かいしゃ) | 회사 | 会社で働きます (회사에서 일합니다) |
| ご飯(はん) | 밥/식사 | 朝(あさ)ご飯 (아침밥) |
| 仕事(しごと) | 일/직업 | 仕事をします (일을 합니다) |
| 日本(にほん) | 일본 | 日本へ行きます (일본에 갑니다) |
| 先生(せんせい) | 선생님 | 先生に会います (선생님을 만납니다) |
| 会(あ)います | 만납니다 | 友達に会います (친구를 만납니다) |
| 行(い)きます | 갑니다 | 学校に行きます (학교에 갑니다) |
