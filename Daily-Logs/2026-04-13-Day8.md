4. この町(まち)は 静(しず)かですね (정답!)

### AI 교정본
```
1. 昨日(きのう)は 少し 寒(さむ)かったです
2. 家族(かぞく)は 四人(よにん)です
3. 天気(てんき)が 良(よ)かったです

4. この町(まち)は 静(しず)かですね
5. 昨日は 暇(ひま)でした
6. 先生(せんせい)は 親切(しんせつ)でした

7. 明日(あした)は 暇(ひま)じゃありません (내일은 한가하지 않습니다)
8. ソウルは 賑(にぎ)やかでした (서울은 북적거렸습니다) - 완벽!
9. 昨日(きのう)は 静(しず)かじゃありませんでした (어제는 조용하지 않았습니다)

10. この 本(ほん)は 有名(ゆうめい)じゃないです (이 책은 유명하지 않습니다) - 완벽!
11. 昨日(きのう)は 元気(げんき)でした (어제는 건강했습니다) - 완벽!
12. その 人(ひと)は 親切(しんせつ)じゃありませんでした (그 사람은 친절하지 않았습니다)
```

### 교정 포인트
1. **문법 오류**: 
   - **핵심**: い형용사와 な형용사의 과거형 구분
   - い형용사 (寒(さむ)い, 良(よ)い): `い` 탈락 + `~かったです`
   - な형용사 (暇(ひま), 親切(しんせつ)): 명사처럼 `~でした` 사용
   - **주의**: な형용사에는 절대 `~かったです`를 붙이지 않습니다! 명사와 동일하게 취급하세요.

2. **부정형과 과거부정형 (な형용사 = 명사)**:
   - 명사: 학생(学生) → 학생이 아닙니다: `学生じゃありません` / 학생이 아니었습니다: `学生じゃありませんでした`
   - な형용사: 한가함(暇) → 한가하지 않습니다: `暇(ひま)じゃありません`
   - な형용사: 조용함(静か) → 조용하지 않았습니다: `静(しず)かじゃありませんでした`
   - 오류 분석: `しずかったでません`은 い형용사의 `~かったです`와 동사의 부정형 `~ません`이 섞인 형태입니다. な형용사는 명사와 똑같이 끝에 `じゃありません(でした)`만 붙이면 됩니다!

3. **칭찬 포인트 (`じゃないです`)**:
   - `じゃありません` 대신 `じゃないです`를 사용하셨는데, 이것은 **완벽하게 맞는 표현**이며 실제 일본인들이 회화에서 훨씬 더 자주 쓰는 아주 자연스러운 표현입니다! 언어 감각이 정말 뛰어나시네요!

## 📝 새로 배운 표현

| 표현 | 의미 | 예문 |
|------|------|------|
| **な형용사 + 입니다(です)** | (현재) ~입니다 | 静(しず)かです (조용합니다) |
| **나형용사 + じゃありません** | (부정) ~하지 않습니다 | 暇(ひま)じゃありません (한가하지 않습니다) |
| **나형용사 + 였습니다(였습니다)** | (과거) ~했습니다 | 親切(しんせつ)でした (친절했습니다) |
| **나형용사 + じゃない입니다** | (부정/구어) ~하지 않습니다 | 有名(ゆうめい)じゃないです (유명하지 않습니다) |
| **ですが / でも** | 하지만 / 그런데 | 難しいですが、楽しいです (어렵지만 즐겁습니다) |
