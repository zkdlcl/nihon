## 💬 대화 내용

### 나의 일본어
```
1. かばんはあります
2. 犬がいます
3. 机の下にいます
4. 家의近く에學校가あります
5. 車の中でかばんがあります
6. コンビニの右に郵便局があります
7. 銀行と湯便局の間にコンビニがあります
```

### AI 교정본
```
1. かばんが あります。
2. 犬(いぬ)が います。 (Perfect!)
3. 机(つくえ)の 下(した)に 猫(ねこ)가 います。 (주어 추가 시)
4. 家(いえ)の 近(ちか)くに 学校(がっこう)が あります。 (Perfect!)
5. 車(くるま)の 中(なか)に かばんが あります。
6. コンビニの 右(みぎ)に 郵便局(ゆうびんきょく)が あります。 (Perfect!)
7. 銀行(ぎんこう)と 郵便局(ゆうびんきょく)の 間(あいだ)に コンビニが あります。
```

### 교정 포인트
1. **문법 오류**: 
   - **조사 に vs で**: 존재를 나타낼 때는 `に`(에)를 사용합니다. (`で`는 동작의 장소)
   - **조사 が vs は**: 기본 존재 표현에서는 `が`(이/가)가 더 자연스럽습니다.

2. **어휘 선택**:
   - **郵便局(ゆうびんきょく)**: '우체국'의 한자는 `郵`입니다. `湯`(뜨거운 물)과 헷갈리지 않도록 주의!

## 📝 새로 배운 표현

| 표현 | 의미 | 예문 |
|------|------|------|
| あります | 있습니다 (무생물) | 本(ほん)が あります。 |
| います | 있습니다 (생물) | 犬(いぬ)가 います。 |
| ~の 上(うえ)/下(した) | ~의 위/아래 | 机(つくえ)の 下(した)に あります。 |
| ~の 前(まえ)/後ろ(うしろ) | ~의 앞/뒤 | 家(いえ)の 前(まえ)に います。 |
| ~の 右(みぎ)/左(ひだり) | ~의 오른쪽/왼쪽 | コンビニの 右(みぎ)に あります。 |
| ~の 間(あいだ) | ~의 사이 | Aと Bの 間(あいだ)に あります。 |
