## 💬 대화 내용

### 나의 일본어
```
1. ごれは何ですか
2. りょこうはいつですか
3. トイレはどこですか
4. そのひとはだれでしか？

5. このかばんはいくらですか
6. りんごはいくつですか
7. 仕事はどうですか
8. どうして大学にいきません

### 어휘 확장 연습 (나의 일본어)
1. 明日、どもたちといっしょに日本に行きます
2. だれはいますか
3. なでもすきです
4. どこでも行きたいですか
```

### AI 교정본
```
1. これは何(なん)ですか？
2. 旅行(りょこう)は いつですか？
3. トイレは どこですか？
4. その人(ひと)は 誰(だれ)ですか？

5. このかばんは いくらですか？
6. りんごは いくつですか？
7. 仕事(しごと)は どうですか？
8. どうして大学(だいがく)に 行(い)きませんでしたか？

### 어휘 확장 연습 (AI 교정본)
1. 明日(あした)、友達(ともだち)と一緒(いっしょ)に日本(にほん)に行(い)きます。
2. 誰か(だれか) いますか？
3. 何でも(なんでも) 好(す)きです。
4. どこか(に) 行(い)きたいですか？ (또는 どこでも 行(い)きたいですか？)
```

### 교정 포인트
1. **발음/타이핑 오류 (`ご` vs `こ`)**: 
   - 오류: ごれ
   - 올바른 표현: これ
   - 설명: '이것'은 탁음이 없는 `これ`입니다. 탁음(`ご`)과 청음(`こ`)을 쓸 때 주의해 주세요!

2. **발음/타이핑 오류 (`し` vs `す`)**:
   - 오류: だれでしか
   - 올바른 표현: だれですか
   - 설명: `です`에 질문형 `か`를 붙이는 것이므로 `ですか`가 맞습니다. `し(shi)`와 `す(su)` 타이핑 오타를 조심하세요.

3. **시제 오류 (현재부정 vs 과거부정)**:
   - 오류: どうして大学にいきません
   - 올바른 표현: どうして大学に 行(い)きませんでしたか
   - 설명: "가지 **않았**습니까?" 라고 과거로 물어보았으므로, `いきません`이 아니라 과거 부정형인 `いきませんでした`를 써야 합니다. 그리고 질문이므로 끝에 `か`를 붙여주세요! (학교 대신 '대학'이라는 단어를 쓴 것은 아주 훌륭한 응용입니다👍)

4. **완벽한 문장 (백점 만점💯)**:
   - 2번, 3번, 5번, 6번, 7번 문장 모두 의문사와 조사를 완벽하게 사용하셨습니다! 어휘력과 응용력이 대단하시네요.

5. **단어 오타 교정 (`도` vs `토` / `난` vs `나`)**:
   - `どもたち` -> `ともだち(友達)`: 친구는 '도모'가 아니라 '토모'입니다. 탁음 없이!
   - `なでも` -> `なんでも`: 뭐든지는 '나데모'가 아니라 '난데모'입니다. 응 발음을 꼭 넣어주세요!

6. **의문사 + か/でも 혼동**:
   - 질문: 누군가 있습니까? / 작성: `だれはいますか`
   - 교정: `だれか いますか？`
   - 설명: 누군가(불특정)를 가리킬 때는 `だれ`에 `か`를 붙여서 `だれか`가 됩니다. `だれは`라고 하면 "누구는 있습니까?"라는 조금 어색한 문장이 됩니다.
   - 질문: 어딘가에 가고 싶습니까? / 작성: `どこでも行きたいですか`
   - 설명: 어딘가(somewhere)는 `どこか`입니다! 작성해주신 `どこでも`는 "어디든지(anywhere)"라는 뜻이 됩니다. "어디든지 가고 싶습니까?"라는 의미로 쓰셨다면 문법적으로 아주 훌륭한 문장입니다!
