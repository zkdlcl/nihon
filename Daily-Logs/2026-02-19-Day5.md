## 📖 오늘의 핵심 문법: ます형

### ます형 그룹 정리
| 그룹 | 특징 | 변환 규칙 | 예시 |
|------|------|----------|------|
| **1그룹** (う동사) | く/ぐ/す/つ/ぬ/ぶ/む/る로 끝남 | 어미 → い단 + ます | 書(か)く → 書(か)きます |
| **2그룹** (る동사) | る로 끝나고 앞이 い/え단 | る 빼고 + ます | 食(た)べる → 食(た)べます |
| **3그룹** (불규칙) | する / 来(く)る만 | 통째로 암기 | する → します |

### ⚠️ 주의! 来る의 ます형
- 기본형: 来(く)る (쿠루)
- ます형: 来(き)ます (키마스) ← 읽는 법이 く→き로 바뀜!
- 부정형: 来(き)ません (키마센)

## 💬 오늘의 연습

### 나의 일본어
```
わたしわほんおよみます
私和英がおみません
わたしわがｃこうにいきます
```

### AI 교정본
```
私はほんをよみます
私はえいがをみません
私はがっこうにいきます
```

### 교정 포인트
1. **조사 오류** (가장 많이 틀린 것!):
   - わ → **は** (주제 조사: 발음은 わ지만 쓸 때는 は)
   - お → **を** (목적 조사: 발음은 お지만 쓸 때는 を / 타이핑: `wo`)

2. **っ 타이핑**:
   - がｃこう → **がっこう** (다음 자음을 2번: `gakkou`)

3. **한자 주의**:
   - 渡し(건네주다) ≠ **私**(나) → 같은 わたし 발음이지만 다른 한자!

## 📝 새로 배운 표현

### ます형 4가지 활용
| 형태 | 표현 | 의미 |
|------|------|------|
| 현재/미래 | たべます | 먹습니다 |
| 현재 부정 | たべません | 먹지 않습니다 |
| 과거 ✨ | たべ**ました** | 먹었습니다 |
| 과거 부정 ✨ | たべ**ませんでした** | 먹지 않았습니다 |

### 오늘 만든 문장 (과거형)
| 문장 | 의미 |
|------|------|
| 私はほんをよみました | 나는 책을 읽었습니다 |
| 私はえいがをみませんでした | 나는 영화를 보지 않았습니다 |
| 私はがっこうにいきました | 나는 학교에 갔습니다 |
