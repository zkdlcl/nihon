# 🇯🇵 JPT 400 학습 로드맵

## 📊 현재 상태
- **목표 점수**: JPT 400점
- **현재 수준**: 히라가나/카타카나 완료, 듀오링고 11-12레벨
- **목표 기간**: 3-6개월 집중 학습
- **대응 레벨**: JLPT N4 수준 (N3 근접)

## 🎯 JPT 400점 요구사항

### 어휘력
- **필요 어휘**: 1,500~2,000개
- **한자**: 300~400자
- **현재 → 목표**: 듀오링고 기본 어휘 → N5 완성 + N4 70%

### 문법
- **N5 문법**: 기본 조사, です/ます형, 기본 시제
- **N4 문법**: ~ている, ~ながら, ~ので, ~たことがある, ~なければならない 등

### 시험 구성
**청해 (Listening) - 100문제**
- Part 1: 사진 묘사
- Part 2: 질의응답
- Part 3: 회화문
- Part 4: 설명문

**독해 (Reading) - 100문제**
- Part 5: 정답 찾기 (어휘/문법)
- Part 6: 오문정정
- Part 7: 공란 메우기
- Part 8: 장문 독해

## 📅 3개월 학습 로드맵

### Month 1: N5 완성 + N4 기초 (Week 1-4)
**Week 1-2: N5 문법 총정리**
- [ ] 조사 완벽 정리 (は, が, を, に, で, へ, と, から, まで)
- [ ] です/ます형 활용
- [ ] 기본 형용사 (い형용사, な형용사)
- [ ] 기본 시제 (과거/현재/미래)
- [ ] 매일 어휘 30개 암기

**Week 3-4: N4 기초 문법**
- [ ] て형 활용 및 ~ている (진행/상태)
- [ ] ~たい, ~たがる (희망)
- [ ] ~ことができる (가능)
- [ ] ~前に, ~後で (시간 표현)
- [ ] 매일 어휘 40개 암기

### Month 2: N4 핵심 + 청해 집중 (Week 5-8)
**Week 5-6: N4 핵심 문법**
- [ ] ~ながら (~하면서)
- [ ] ~ので, ~から (이유)
- [ ] ~たことがある (경험)
- [ ] ~なければならない (의무)
- [ ] ~そうだ, ~ようだ (추측/전문)
- [ ] 매일 어휘 40개 암기

**Week 7-8: 청해 훈련**
- [ ] Part 1-2 집중 (기본 듣기)
- [ ] 듀오링고 듣기 연습 매일 30분
- [ ] 일본어 팟캐스트/유튜브 (초급) 매일 20분
- [ ] 매일 어휘 복습 30개

### Month 3: 실전 대비 + 독해 강화 (Week 9-12)
**Week 9-10: 독해 집중**
- [ ] Part 5-6 문법/어휘 문제 풀이
- [ ] Part 7 공란 메우기 연습
- [ ] Part 8 장문 독해 (시간 배분 연습)
- [ ] 매일 독해 지문 2-3개

**Week 11-12: 실전 모의고사**
- [ ] 모의고사 1회차 (약점 파악)
- [ ] 약점 집중 보완
- [ ] 모의고사 2-3회차
- [ ] 최종 어휘/문법 총정리

## 📚 일일 학습 루틴 (평일 기준)

### 기본 루틴 (1-2개월차)
- **아침 (30분)**: 어휘 암기 + 복습
- **점심 (20분)**: 듀오링고 + 문법 복습
- **저녁 (40분)**: 새 문법 학습 + 예문 작성
- **자기 전 (20분)**: 청해 연습 or 일본어 컨텐츠

### 실전 루틴 (3개월차)
- **아침 (30분)**: 모의고사 Part 풀이
- **점심 (20분)**: 오답 정리
- **저녁 (50분)**: 약점 보완 + 독해 연습
- **자기 전 (20분)**: 청해 연습
