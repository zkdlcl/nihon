# 🎓 일본어 학습 세션 시작 가이드

## 📌 빠른 시작

매일 학습을 시작할 때 이렇게 말하세요:
```
"일본어 공부 시작" 또는 "Day {N} 시작"
```

AI가 자동으로:
1. ✅ 오늘의 학습 계획 확인
2. ✅ Daily Log 생성
3. ✅ 학습 내용 안내
4. ✅ 대화형 수업 진행

---

## 🗓️ 학습 세션 구조

### 1단계: 워밍업 (5분)
- 어제 배운 내용 복습
- 오늘의 목표 확인

### 2단계: 문법 학습 (20분)
- 새로운 문법 설명
- 예문 분석
- 연습 문제

### 3단계: 어휘 학습 (20분)
- 새 단어 30개 학습
- 예문과 함께 암기
- 발음 연습

### 4단계: 실전 연습 (15분)
- 예문 작성
- AI와 대화 연습
- 즉시 피드백

### 5단계: 마무리 (10분)
- 오늘 배운 내용 정리
- Daily Log 작성
- 내일 계획 확인

**총 소요 시간**: 약 70분

---

## 💡 학습 팁

### 효과적인 질문 방법
```
❌ "이거 뭐야?"
✅ "は와 が의 차이를 예문으로 설명해줘"

❌ "단어 알려줘"
✅ "가족 관련 명사 10개를 예문과 함께 알려줘"

❌ "이거 맞아?"
✅ "이 문장을 교정해줘: 私が学生です"
```

### AI 활용 방법
- **문법 설명**: "~ている 문법을 초보자도 이해할 수 있게 설명해줘"
- **예문 요청**: "~ながら를 사용한 예문 5개 만들어줘"
- **교정 요청**: "이 문장 교정해줘: [내 문장]"
- **퀴즈**: "오늘 배운 문법으로 퀴즈 5개 내줘"
